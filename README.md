# 📝 Lista de Tarefas Simples - SQLite

Um app **MUITO SIMPLES** para ensinar React Native + SQLite.

## 🎯 O que faz?

- ✅ Adicionar tarefas
- ✏️ Marcar como feitas
- 🗑️ Excluir tarefas
- 💾 Salva no celular

## � Como usar?

1. **Adicionar**: Digite e toque no "+"
2. **Marcar**: Toque na tarefa
3. **Excluir**: Segure a tarefa
